## Installation

Software necesario:
- node
- npm
- git

### Clonar

- Clonar este repositorio `https://github.com/fvcproductions/SOMEREPO`

### Setup

```shell
$ npm install
```

> Crear un archivo .env en la carpeta raiz y agregar API KEY de mongo DB

Iniciar en modo development

```shell
$ npm run start
```
